# Google Cloud
We need to enable the text-to-speech API.

1. Log into the Google Cloud
2. Select an existing project, or create a new project. The project must have billing enabled.
3. In the Search for resources box, type speech
4. Select "Cloud Text-to-Speech API"
5. Enable the API
6. Navigate to APIs & services -> Credentials
7. Select an existing Service Account or create a new Service Account (No permissions or pricipals need to be added).
8. We will create a new key during the installation.
